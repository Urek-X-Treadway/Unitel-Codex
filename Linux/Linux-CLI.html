📁 File Management

  • ls – List directory contents
  • cd – Change directory
  • pwd – Print current directory path
  • cp file1 file2 – Copy file1 to file2
  • mv old new – Move or rename files
  • rm file – Remove file
  • mkdir folder – Create directory
  • rmdir folder – Delete empty directory
  • touch file – Create empty file
  • stat file – Show file info (size, timestamps, permissions)
  • basename path – Get filename from path
  • dirname path – Get directory name from path

🔧 System Information

  • uname -a – Show complete system info (kernel, architecture, hostname)
  • hostname – Display current system hostname
  • uptime – Show how long the system has been running
  • top – Display real-time CPU and memory usage
  • htop – Enhanced interactive process viewer (if installed)
  • df -h – Disk space usage (human-readable)
  • du -sh * – Folder sizes in current directory
  • free -m – Show memory usage in MB
  • lscpu – CPU architecture and info
  • lsblk – List block devices (disks and partitions)
  • dmesg – Show system boot and hardware messages

👤 User Management

  • whoami – Show current logged-in user
  • id – Show UID, GID, and groups of user
  • users – Show all users currently logged in
  • adduser username – Add a new user
  • passwd username – Set/change user password
  • usermod -aG group username – Add user to group
  • groups username – Show groups a user belongs to
  • deluser username – Delete a user
  • who – Show who is logged in and from where
  • su - username – Switch to another user account

🌐 Networking

  • ping google.com – Check if host is reachable
  • ip a – Show IP addresses and interfaces
  • ifconfig – Old tool for network interfaces (still used)
  • netstat -tulnp – List listening ports and services
  • ss -tuln – Faster replacement for netstat
  • curl ifconfig.me – Get public IP address
  • traceroute domain.com – Show route packets take
  • dig example.com – DNS lookup
  • nslookup example.com – Query name servers
  • nmap 192.168.1.1 – Scan ports on a target (requires install)

📦 Package Management (Debian/Ubuntu)

  • apt update – Refresh package lists
  • apt upgrade – Upgrade all installed packages
  • apt install <package> – Install a package
  • apt remove <package> – Remove a package
  • apt search <package> – Search for packages
  • dpkg -i <file.deb> – Install .deb package file manually
  • apt autoremove – Remove unused dependencies
  • apt clean – Clear local repository of retrieved package files

🔄 Processes & Services

  • ps aux – Show all running processes
  • top – Interactive real-time process viewer
  • htop – Enhanced process viewer (if installed)
  • kill <PID> – Terminate process by PID
  • killall <process_name> – Terminate all processes with name
  • systemctl status <service> – Show service status (systemd)
  • systemctl start <service> – Start a service
  • systemctl stop <service> – Stop a service
  • systemctl restart <service> – Restart a service
  • service <service> start|stop|restart – Control services (SysV init)

🧪 Disk & Filesystems

  • df -h – Show disk space usage in human-readable format
  • du -sh <directory> – Show total size of directory
  • mount | umount – Mount or unmount filesystems
  • lsblk – List block devices (disks and partitions)
  • fdisk -l – Show partition tables
  • blkid – Show block device attributes like UUID and type
  • tune2fs – Adjust ext2/3/4 filesystem parameters
  • fsck /dev/sdX – Filesystem consistency check and repair

🔐 Permissions & Ownership

  • chmod 755 file – Set file permissions (rwxr-xr-x)
  • chown user:group file – Change owner and group
  • ls -l file – List file with permissions and ownership
  • umask – Show default permission mask for new files
  • getfacl file – Get extended ACL permissions
  • setfacl -m u:user:rwx file – Set extended ACL permissions

🔍 Search & Filters

  • grep 'pattern' file – Search text in files
  • find /path -name 'file' – Find files by name
  • locate file – Quickly find file by name (updatedb index)
  • awk '{print $1}' file – Pattern scanning and processing
  • sed 's/old/new/g' file – Stream editor for text substitution
  • cut -d',' -f1 file – Extract columns from file
  • sort file – Sort lines alphabetically or numerically
  • uniq file – Filter duplicate lines

🛠️ Bash & Scripting

  • echo "Hello World" – Print text to terminal
  • #!/bin/bash – Shebang to specify script interpreter
  • chmod +x script.sh – Make script executable
  • ./script.sh – Run executable script
  • if [ condition ]; then ... fi – Basic if statement
  • for i in {1..5}; do ... done – For loop
  • while [ condition ]; do ... done – While loop
  • read variable – Read input from user
  • $? – Exit status of last command
  • export VAR=value – Set environment variable
